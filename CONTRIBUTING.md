# Contributing

Thanks for helping improve Clawguard.

## Quick start

1. Fork the repo and create a feature branch.
2. Run `npm install`.
3. Run tests with `npm test`.
4. Open a PR with a clear description and screenshots if relevant.

## Guidelines

- Keep changes focused and avoid unrelated refactors.
- Add tests for new rules or behaviors when possible.
- Keep docs updated for user-facing changes.

## Reporting security issues

Please use `SECURITY.md` for security disclosures.
